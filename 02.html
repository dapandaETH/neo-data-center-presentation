Project Overview

Infrastructure Transition

Transitioning from BTC mining to LLM inference infrastructure, leveraging existing hardware investments while pivoting to high-value computational workloads.

⚡

Open-Weight Models

Running open-weight language models for customers, including Llama 3 70B, Mistral 7B, and multimodal models, providing managed API endpoints with pay-per-token pricing.

◈

Hybrid Approach

Hybrid approach: Strategic mix of reusing existing GPUs and deploying separate infrastructure for different workloads, optimizing cost-efficiency and operational flexibility.

◎

Target Scale

Medium-scale deployment: 10-100 GPUs for LLM inference operations, supporting both high-throughput batch processing and low-latency interactive requests.
