OpenAI v1 API Compatibility

Drop-in replacement for OpenAI API with identical request/response format

Request Format 

{
  "model": "llama3-70b-instruct",
  "messages": [
    {
      "role": "system",
      "content": "You are a helpful assistant"
    },
    {
      "role": "user",
      "content": "Hello!"
    }
  ],
  "temperature": 0.7,
  "max_tokens": 1024,
  "top_p": 0.9,
  "stream": false
}

Parameters

model - Model identifier
messages - Conversation history
temperature - Sampling randomness (0-2)
max_tokens - Max generation length

Response Format 

{
  "id": "chatcmpl-abc123",
  "object": "chat.completion",
  "created": 1715623410,
  "model": "llama3-70b-instruct",
  "choices": [
    {
      "index": 0,
      "message": {
        "role": "assistant",
        "content": "Hello! I'm here to help you."
      },
      "finish_reason": "stop"
    }
  ],
  "usage": {
    "prompt_tokens": 15,
    "completion_tokens": 10,
    "total_tokens": 25
  }
}

Response Fields

id - Unique completion ID
choices - Generated responses
usage - Token usage metrics

HTTP Error Codes

Code Status Description Action
200 OK Request successful Process response
400 Bad Request Invalid parameters or malformed request Fix request format
401 Unauthorized Invalid or missing API key Check authentication
429 Too Many Requests Rate limit exceeded Implement backoff
500 Internal Error Server-side error (retryable) Retry with backoff
503 Service Unavailable Model unavailable or loading Retry after delay
504 Gateway Timeout Request timeout exceeded Reduce max_tokens

Note: Compatible with OpenAI SDK, LangChain, LlamaIndex, and any tool using the OpenAI v1 API specification.
